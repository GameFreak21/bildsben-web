About Me

Writing Projects

Writing is much less complicated than coding. If you want a written piece, you can choose for me to do the research independently, where the turnaround for a project will be about 3 days for a ~700 word piece. This will cost £20. This time period can be shortened to same day* for an extra fee. All references used will be harvard style references.


You can also give me research to build a piece on. This works by sending me a document with all the research already done, with links/references given, and I will create a written piece based on the given research. The turnaround for this sort of project is 2 days for a ~700 word piece. This will cost £15. You can also have same day* return of the project for a fee. All references used will be harvard style references.


Aditionally, I can work collaboratively with other writers to create a piece. The time period for my share of work is roughly 2 days for a ~400 word segment. This will cost £10. You can also have same day* return of the project for a fee. All references used will be harvard style references.


* - same day return only applicable if project was requested, recieved and discussed BEFORE 2PM Europe/London time. If requested after 2pm, the delivery for the project will be next day.
